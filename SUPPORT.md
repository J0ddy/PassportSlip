# Support

## Where to get help

Use the most appropriate channel:

| Need | Channel |
|---|---|
| Bug report | GitHub issue |
| Feature request | GitHub issue |
| Security vulnerability | `SECURITY.md` |
| Commercial/project enquiry | `{{CONTACT_EMAIL}}` |
| Documentation issue | GitHub issue or pull request |

## Before opening an issue

Please include:

- project version or commit SHA
- operating system
- runtime version
- package manager and version
- exact command run
- expected behaviour
- actual behaviour
- relevant logs or screenshots

Do not include secrets, tokens, credentials, private keys, customer data, or personal data.

## Response expectations

This project may not provide guaranteed support unless there is a separate written support agreement.
